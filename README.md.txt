# COVID-19 Data Analysis using PySpark

## Project Overview
This project analyzes COVID-19 data using PySpark.

## Tasks Performed
- Loaded COVID-19 dataset
- Filtered country-specific data
- Found total cases and deaths
- Sorted countries by highest cases

## Tools Used
- Python
- PySpark
- Jupyter Notebook
- Anaconda

## Dataset
OWID COVID-19 Dataset

## Output
The project displays:
- COVID case records
- Total deaths and cases
- Highest affected countries
